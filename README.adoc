== Add my own settings
1. Create a weasel.custom.yaml
2. Open *weasel.custom.yaml* add the following snippets

[source, yaml]
----
patch:
    "style/+":
        __include: "weasel_settings:/style"
    "preset_color_schemes/+":
        __include: "weasel_settings:/preset_color_schemes"
----
